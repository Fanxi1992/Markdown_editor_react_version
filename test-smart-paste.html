验证要点：
  • 富文本内容应自动转换为 Markdown
  • 纯 Markdown 文本应直接粘贴，不重复转换
  • 普通纯文本应正常粘贴
  • 转换时应显示提示消息
  • 格式应保持整洁，避免多余空行
